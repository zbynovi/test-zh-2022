Nadpis druhé úrovně k vyznačení ručním zápisem tagů

Odstavec k přesunutí do značek vložených pomocí ctrl+mezera → adresa ke zkopírování: https://validator.w3.org → odkaz k vyznačení pomocí výběru a následného doplnění značkami pomocí ctrl+mezera.

Cvičení 3 v 1 - chybu pochop, najdi a oprav

  1. Consider adding a lang attribute to the html start tag to declare the language of this document.

    Doporučuje se vždy deklarovat přirozený jazyk, kterým je psán obsah webové stránky, a to připojením atributu lang="cs" do počáteční značky elementu html.

  2. The character encoding was not declared.

    Abychom předešli problémům s nekorektním zobrazením znaků, vkládáme do hlavičky meta-element s informací o použité tabulce kódování znaků: <meta charset="UTF-8">

  3. No DOCTYPE found!

    Na začátek dokumentu patří deklarace verze použitého jazyka: <!DOCTYPE html>

  4. Bad character…

    Některé znaky (např. <>) se speciálními významy v kódování HTML musí být vkládány jako entity (tj. sekvence s kódem uzavřená mezi ampersand a středník).

  5. Element head is missing a required instance of child element title.

    V hlavičce nesmí chybět titulek (byť by byl prázdný či nesmyslný).

  6. Element … not allowed as child of element … in this context

    Nejsou dovoleny některé kombinace rodičovského elementu a elementu do něj vloženého, pokud to odporuje očekávanému druhu vkládaného obsahu - např. blokový prvek nesmí být obsahem řádkového prvku.

  7. End tag … violates nesting rules.

    Značky vnořených párových elementů se nesmí křížit - koncové značky jsou uspořádány v opačném pořadí než počáteční.

  8. The … element is obsolete. Use CSS instead

    Použití zastaralého elementu (např. s ryze formátovacím významem, což byla v počátcích HTML běžná praxe). Využívejte odpovídající elementy s vhodným věcným významem a formátujte výhradně pomocí kaskádových stylů.

  9. An img element must have an alt attribute

    Obrázku chybí povinný atribut s alternativním textem.

  10. Attribute … not allowed on element … at this point

    Má-li atribut víceslovnou hodnotu, která není uzavřená do uvozovek nebo apostrofů, bude prohlížečem chybně rozeznána jako další atribut. Hodnota atributu by měla být do uvozovek/apostrofů vždy uzavřena.
